Adaptive Dopamine Regulation System

A lightweight Python-based productivity app that helps users manage screen time and improve focus using behavioral feedback.

Features

*  Track daily screen time
*  Energy level-based suggestions (Low / High)
*  Streak tracking system
*  Smart activity recommendations
*  Simple GUI using Tkinter

 Tech Stack

* Python
* Tkinter (GUI)
* JSON (data storage)


Future Improvements

* Focus timer (Pomodoro)
* Notifications
* Mobile app version
* Data visualization dashboard
