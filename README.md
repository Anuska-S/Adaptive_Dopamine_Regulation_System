BetterHabits – Screen Time Management App

BetterHabits is a Flask-based web application designed to help users manage their screen time through a structured, reward-based system. It encourages healthier digital habits by tracking usage, setting limits, and suggesting alternative activities.


Features

- Track daily screen time
- Set usage limits and daily goals
- Earn points for maintaining healthy screen habits
- Activity suggestions based on user energy levels
- Custom settings (theme, focus timer, break timer)
- Weekly progress tracking


Tech Stack

- Backend:Python (Flask)
- Frontend:HTML, CSS
- Storage:JSON (lightweight data handling)
- Tools:Git, GitHub, VS Code
